What are the different ways to select an element in the DOM?

1. "document.getElementById" - Select an element by its ID.

2. "document.getElementsByClassName" - Select elements by their class name.

3. "document.getElementsByTagName" - Select elements by their tag name.

4. "document.querySelector" - Select the first element that matches a CSS selector.

5. "document.querySelectorAll" - Select all elements that match a CSS selector.

What is the difference between innerHTML, innerText and textContent?

- innerHTML: Returns the HTML content of an element, including any child elements.

- innerText: Returns the visible text content of an element, excluding any HTML tags.

- textContent: Similar to innerText, but returns the text content of all elements, including hidden ones.

What is event delegation in the DOM?

Event delegation is a technique in which a single event listener is added to a parent element instead of individual child elements. This allows the parent to handle events for all its children, improving performance and reducing memory usage. When an event occurs, it bubbles up to the parent, which can then determine the target element and respond accordingly.

What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and bubbles up to its parent node.
